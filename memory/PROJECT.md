# Project Memory - Global Context

## Project Overview

### Project Name
HairWeb - Landing Page Peluquería

### Project Description
Landing page estática en HTML puro para una peluquería. Página única informativa con servicios, precios, ubicación y contacto.

### Project Goals
1. Presentar los servicios de la peluquería
2. Facilitar el contacto con clientes potenciales
3. Mostrar ubicación y horarios

### Quality Rules
1. HTML válido (W3C)
2. Etiquetas semánticas correctas
3. Atributos alt en todas las imágenes
4. Estructura responsive con meta viewport
